Minimum Window Substring

Find the minimum window in source string containing all characters of target string

leetcode 76 | hard

Algorithm Theory

Minimum Window Substring is a classic hard application of sliding window algorithm. Find the minimum contiguous substring in source string that contains all characters (including duplicates) of target string.

Algorithm Steps:

  1. Preprocessing: Count frequency of each character in target string
  2. Initialize: Both left and right pointers start at beginning, create window character frequency counter
  3. Expand Window: Move right pointer forward, add new character to window and update frequency
  4. Check Coverage: Determine if current window contains all characters of target string
  5. Shrink Window: When window satisfies condition, try moving left pointer to find smaller window
  6. Update Minimum: Record the minimum window that satisfies condition
  7. Continue: Repeat steps 3-6 until right pointer reaches end

Time Complexity

O(|s| + |t|)

where |s| and |t| are lengths of source and target strings

Space Complexity

O(|s| + |t|)

Additional hash tables needed to store character frequencies

Code Implementation

function minWindow(s, t) {
    if (s.length < t.length) return "";

    // Count frequency of target string characters
    const targetFreq = new Map();
    for (const char of t) {
        targetFreq.set(char, (targetFreq.get(char) || 0) + 1);
    }

    let left = 0;
    let right = 0;
    let minLen = Infinity;
    let minStart = 0;
    let requiredChars = targetFreq.size;
    let formedChars = 0;
    const windowFreq = new Map();

    while (right < s.length) {
        // Expand window
        const rightChar = s[right];
        windowFreq.set(rightChar, (windowFreq.get(rightChar) || 0) + 1);

        // Check if frequency requirement is satisfied for a character
        if (targetFreq.has(rightChar) &&
            windowFreq.get(rightChar) === targetFreq.get(rightChar)) {
            formedChars++;
        }

        // Try to shrink window
        while (left <= right && formedChars === requiredChars) {
            const leftChar = s[left];

            // Update minimum window
            if (right - left + 1 < minLen) {
                minLen = right - left + 1;
                minStart = left;
            }

            // Remove left character
            windowFreq.set(leftChar, windowFreq.get(leftChar) - 1);
            if (targetFreq.has(leftChar) &&
                windowFreq.get(leftChar) < targetFreq.get(leftChar)) {
                formedChars--;
            }

            left++;
        }

        right++;
    }

    return minLen === Infinity ? "" : s.substring(minStart, minStart + minLen);
}
