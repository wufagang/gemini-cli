Algorithm Overview

滑动窗口算法是一种高效的数组/字符串处理技术，通过维护一个窗口在数据结构上滑动来解决子数组/子字符串问题。核心思想是避免重复计算，将暴力解法的 O(n²) 或 O(n³) 时间复杂度优化到 O(n)。

算法分为固定窗口和可变窗口两种模式。固定窗口适用于窗口大小恒定的问题；可变窗口根据条件动态调整大小，通常使用双指针技术实现。

掌握这些模式后，能够快速识别和解决相关问题，在处理大规模数据时具有显著的性能优势。
