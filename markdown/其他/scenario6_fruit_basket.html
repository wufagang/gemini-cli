Fruit Into Baskets

Variable window sliding algorithm for longest subarray with at most two distinct elements

leetcode 904 | medium

Algorithm Theory

Fruit Into Baskets finds the longest subarray with at most two distinct types. Demonstrates type constraint management and hash map optimization strategies.

Algorithm Steps:

  1. Initialize: Use sliding window with hash map to track fruit types
  2. Expand Window: Add fruits while maintaining at most 2 types
  3. Shrink Window: When exceeding 2 types, shrink from left
  4. Update Maximum: Track longest valid subarray length

Code Implementation

function totalFruit(fruits) {
    const fruitCount = new Map();
    let left = 0;
    let maxLength = 0;

    for (let right = 0; right < fruits.length; right++) {
        // Add current fruit to window
        fruitCount.set(fruits[right], (fruitCount.get(fruits[right]) || 0) + 1);

        // Shrink window if more than 2 types
        while (fruitCount.size > 2) {
            const leftFruit = fruits[left];
            fruitCount.set(leftFruit, fruitCount.get(leftFruit) - 1);
            if (fruitCount.get(leftFruit) === 0) {
                fruitCount.delete(leftFruit);
            }
            left++;
        }

        // Update maximum length
        maxLength = Math.max(maxLength, right - left + 1);
    }

    return maxLength;
}
