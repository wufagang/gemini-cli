Fixed Window Duplicate Detection

Fixed window size sliding window algorithm for detecting duplicate characters

foundation | easy

Algorithm Theory

Fixed Window Duplicate Detection checks for duplicate characters within a fixed-length window. This is the foundational concept demonstrating basic sliding window principles.

Algorithm Steps:

  1. Initialize: Set window size and starting position
  2. Check Window: Examine characters within current window
  3. Detect Duplicates: Use hash set to find repeated characters
  4. Slide Window: Move window one position right
  5. Repeat: Continue until reaching string end

Code Implementation

function hasFixedWindowDuplicates(str, windowSize) {
    const results = [];

    for (let i = 0; i <= str.length - windowSize; i++) {
        const window = str.substring(i, i + windowSize);
        const charSet = new Set();
        let hasDuplicate = false;

        for (const char of window) {
            if (charSet.has(char)) {
                hasDuplicate = true;
                break;
            }
            charSet.add(char);
        }

        results.push({
            position: i,
            window: window,
            hasDuplicate: hasDuplicate
        });
    }

    return results;
}
