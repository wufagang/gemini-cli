Longest Substring Without Repeating Characters

Variable window size sliding window algorithm implementation

leetcode 3 | medium

算法原理

无重复字符的最长子串是滑动窗口算法的经典应用。使用两个指针维护一个无重复字符的窗口，动态调整窗口大小来寻找最长子串。

算法步骤：

  1. 初始化：左右指针都从起始位置开始，创建字符集合记录窗口内字符
  2. 扩展窗口：右指针向前移动，将新字符加入窗口
  3. 检查重复：如果新字符在窗口中存在，则发现重复
  4. 收缩窗口：当出现重复时，左指针右移直到无重复字符
  5. 记录最大值：每次扩展后更新最长子串长度和位置
  6. 继续执行：重复步骤2-5直到右指针到达字符串末尾

时间复杂度

O(n) - n为字符串长度

每个字符最多访问两次（添加和删除）

空间复杂度

O(min(m,n)) - m为字符集大小

需要额外的集合存储窗口内字符

function lengthOfLongestSubstring(s) {
    const charSet = new Set();
    let left = 0;
    let maxLength = 0;
    let maxStart = 0;
    let maxEnd = 0;

    for (let right = 0; right < s.length; right++) {
        // If right pointer character is in window, shrink left boundary
        while (charSet.has(s[right])) {
            charSet.delete(s[left]);
            left++;
        }

        // Add right pointer character to window
        charSet.add(s[right]);

        // Update longest substring information
        if (right - left + 1 > maxLength) {
            maxLength = right - left + 1;
            maxStart = left;
            maxEnd = right;
        }
    }

    return {
        length: maxLength,
        substring: s.substring(maxStart, maxEnd + 1),
        start: maxStart,
        end: maxEnd
    };
}
