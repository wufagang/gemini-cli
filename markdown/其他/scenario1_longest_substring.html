Longest Substring Without Repeating Characters

Variable window size sliding window algorithm implementation

leetcode 3 | medium

Algorithm Theory

Longest Substring Without Repeating Characters is a classic application of the sliding window algorithm. It uses two pointers to maintain a window without duplicate characters, dynamically adjusting the window size to find the longest substring.

Algorithm Steps:

  1. Initialize: Both left and right pointers start at the beginning, create a character set to track window characters
  2. Expand Window: Move right pointer forward, add new character to window
  3. Check Duplicates: If new character exists in window, duplicate found
  4. Shrink Window: When duplicate occurs, move left pointer right until no duplicates
  5. Record Maximum: Update longest substring length and position after each expansion
  6. Continue: Repeat steps 2-5 until right pointer reaches string end

Time Complexity

O(n) - where n is string length

Each character visited at most twice (add and remove)

Space Complexity

O(min(m,n)) - where m is character set size

Additional set needed to store window characters

Code Implementation

function lengthOfLongestSubstring(s) {
    const charSet = new Set();
    let left = 0;
    let maxLength = 0;
    let maxStart = 0;
    let maxEnd = 0;

    for (let right = 0; right < s.length; right++) {
        // If right pointer character is in window, shrink left boundary
        while (charSet.has(s[right])) {
            charSet.delete(s[left]);
            left++;
        }

        // Add right pointer character to window
        charSet.add(s[right]);

        // Update longest substring information
        if (right - left + 1 > maxLength) {
            maxLength = right - left + 1;
            maxStart = left;
            maxEnd = right;
        }
    }

    return {
        length: maxLength,
        substring: s.substring(maxStart, maxEnd + 1),
        start: maxStart,
        end: maxEnd
    };
}
