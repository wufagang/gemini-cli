固定窗口重复检测

固定窗口大小滑动窗口算法用于检测重复字符

foundation | easy

算法原理

固定窗口重复检测在固定长度窗口内检查重复字符。这是展示滑动窗口基本原理的基础概念。

算法步骤：

  1. 初始化：设置窗口大小和起始位置
  2. 检查窗口：检查当前窗口内的字符
  3. 检测重复：使用哈希集合找到重复字符
  4. 滑动窗口：将窗口向右移动一个位置
  5. 重复执行：继续直到到达字符串末尾

代码实现

function hasFixedWindowDuplicates(str, windowSize) {
    const results = [];

    for (let i = 0; i <= str.length - windowSize; i++) {
        const window = str.substring(i, i + windowSize);
        const charSet = new Set();
        let hasDuplicate = false;

        for (const char of window) {
            if (charSet.has(char)) {
                hasDuplicate = true;
                break;
            }
            charSet.add(char);
        }

        results.push({
            position: i,
            window: window,
            hasDuplicate: hasDuplicate
        });
    }

    return results;
}
